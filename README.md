# SAIYASASTRA_2
**Horror Game**

--------------------------
**Control**
- Button [ I ] : Open Inventory 
- Button [ N ] : Open Note
- Button [ C ] : Open Craft
- Button [ E ] : Use Item, Open Door
- Mouse Right : Aim
- Mouse Left : Shoot / Use Item
--------------------------------------------
**Craft Menu**
- Drag/Drop Item (Click icon only)
- Mouse Right : Remove Item
