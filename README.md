# SAIYASASTRA_2
**Horror Platformer 2.5D**

--------------------------
**Control**
- Button [ I ] : Open Inventory 
- Button [ N ] : Open Note
- Button [ C ] : Open Craft
- Button [ E ] : Use Item, Open Door
- Mouse Right : Aim
- Mouse Left : Shoot / Use Item
--------------------------------------------
**Craft Menu**
- Drag/Drop Item (Click icon only)
- Mouse Right : Remove Item

**Item Code**
- 000010000 : ขวดน้ำมนต์
- 000131000 : หม้อผี
- 000212000 : ยันต์
- 010101010 : ขวดน้ำมนต์
